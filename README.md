# 🍔 FoodieHub

FoodieHub is a responsive Food Delivery Web Application built using React JS.  
It allows users to browse food items, add them to cart, and manage orders with a clean and modern UI.

---

## 🚀 Features

- Browse food items
- Add to cart / Remove from cart
- Dynamic cart updates
- Responsive design
- Component-based architecture

---

## 🛠 Tech Stack

- React JS
- JavaScript (ES6)
- CSS
- React Router

---

## 📦 Installation

1. Clone the repository
   git clone https://github.com/yourusername/foodiehub.git

2. Install dependencies
   npm install

3. Start the app
   npm start
